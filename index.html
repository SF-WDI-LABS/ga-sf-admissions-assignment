Using the JavaScript Console

The JavaScript Console is a place where we can give the browser specific JavaScript commands and see the output. Assuming you're using a Chrome browser, navigate to View -> Developer -> JavaScript Console to open the console.

Changing an element's CSS class using JavaScript

HTML elements are the content that make up a webpage. We create them with HTML tags < and >. For example the h1 elment is used to be the main heading of a webpage. We could create one in an HTML file by writing <h1>I'm an h1 element</h1>.

If you navigate to example.com, the text "Example Domain" is contained within a h1 element. See for yourself, by right clicking on that text, to open a menu, and selecting the option Inspect.

In the JavaScript console you'll see the word document references the entire HTML page on the screen. It is a variable that stores information about our page that let's us view and manipulate it. This variable is also refered to as the Document Object Model or DOM for short. Using the DOM we can affect the HTML document (webpage) we are viewing.

Let's use the DOM to select something on the webpage. Below there is a set of Pokémon. Let's explore how we could select one of them. They've each been assigned an HTML id that corresponds to their pokémon name. Given an id is unique, it is a good identifier to target something based on. The way we can do this is by using one of the DOM's built-in methods, getElementById.

Note: a method is a function that is contained by an object (in this case, the DOM).

Now, let's catch some Pokémon!

In the JavaScript console try selecting Charmander and removing him (putting him into our pokéball) with the following command: 

        var charmander = document.getElementById("charmander");
charmander.remove();
Did Charmander disapear from the page? If so, good!

Note: refreshing the page resets it to it's origional state, so Charmander reappears. That's because we never changed the content of the HTML file, but simply affected it using JavaScript it after it was loaded.

Input & Output

Today we'll doing I/O in the context of the browser by capturing the user's input to perform an action that outputs a result in the JavaScript console. But what does this mean?

First of all, input/output, also known simply as I/O, is the communication process between the outside world, the input, and a computer, the output. More specifically, inputs can be signals or data recieved by the computer and output can be signals or data sent from it. Examples of input devices include: keyboards, mice, and cameras, while examples of output devices include: monitors, printers, and speakers.

Receiving user input using prompts

As mentioned previously, we're going to write some Javascript code to prompt our user.

Now we're ready to write our prompt message! In the console type:

        var language = "javascript";
function code(lang) {
  return "let's code some " + lang + "!";
}
code(language);

At this point your screen should like something like this, and don't worry if the console is at the bottom or the side of the window.

Great! Now let's prompt the user for some input and see the result displayed in the console. Execute the code below by clicking on the button.

        var sentiment = prompt("How are you feeling today?");
console.log("Today you are", sentiment);

Try clicking the above button. It should produce a prompt message that will display what it's input to the console as output.

Note: console.log() is just a simple way to print something to the console.

Changing the inner HTML of an element to display an output to the user

Now that we can get user input and select elements on a page, we can think about how to display a user's input back to them! We'll have to change what's inside of an HTML element first by selecting it, then setting its innerHTML to anything we want. Let's do this to update the text below in order to reflect something we've learned today.

Today I learned: ...

The elipse above is wrapped in an HTML span tag that has the id "new-knowledge". Knowing this, we should be able to easily select it. Thereafter, we can change what's inside of it to reflect something we've learned. Let's see an example of how this will work.

            var newKnowledge = document.getElementById("new-knowledge");
newKnowledge.innerHTML = "how to select and manipulate DOM elements!"
