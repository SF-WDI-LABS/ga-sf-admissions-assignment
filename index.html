Receiving user input using prompts

We're going to write some Javascript code to prompt our user. We could write the code the file assets/js/app.js, but let's first play around in the JavaScript Console to ensure it's working. Assuming a Chrome browser, to enter into the console, navigate to View -> Developer -> JavaScript Console. Once open, the console should look something like this:

Great! Now we're ready to write our prompt message! In the console type:

        var sentiment = prompt("How are you feeling today?");
        console.log("Today you are", sentiment, "!");

You should have seen a prompt message:

And, depending on what you typed, a corresponding output in the terminal:

Note: console.log() is a simple function that prints what you give it to the console in your browser. You will also notice the last thing the console prints is undefined. That's because console.log() doesn't "return" anything. Don't worry about these details for now!

Practice: Breakfast Prompt

Prompt the user with a message "What did you eat for breakfast this morning?" and display their answer to the console. For example, if the user answered "oatmeal", display "You ate oatmeal for breakfast."

Note: an example solution is provided; please try on your own before you look at it.

Example solution 
        var food = prompt("What did you eat for breakfast this morning?");
        console.log("You ate", food, "for breakfast.");

Selecting HTML elements in JavaScript

HTML elements are the content that make up a webpage. We create them with HTML tags < and >. For example the h1 elment is used to be the main heading of a webpage. We could create one in an HTML file by writing <h1>I'm an h1 element</h1>.

If you navigate to example.com, the text "Example Domain" is contained within a h1 element. See for yourself, by right clicking on that text, to open a menu, and selecting the option Inspect.
